# Diabetes-Prediction
Analysing the Pima Indians Diabetes Dataset using Python and Jupyter Notebook.

## About the Dataset 
The Pima Indian Diabetes Dataset has been used in this study, provided by the UCI Machine Learning Repository. The dataset has been originally collected from the National Institute of Diabetes and Digestive and Kidney Diseases. The dataset consists of some medical distinct variables, such as pregnancy record, BMI, insulin level, age, glucose concentration, diastolic blood pressure, triceps skin fold thickness, diabetes pedigree function etc. This dataset has 768 patientâ€™s data where all the patients are female and at least 21 years old.
